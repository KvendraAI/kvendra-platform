# kvendra-platform

> **Pre-launch.** Multi-tenant KB engine + API + MCP server for the Kvendra platform.

## What lives here

- KB v3-style entity engine (PRJ, CMP, IF, REQ, TEST, ISSUE, REL, ROAD, ADR, PAT, GLO, STD, RUN, UX, DOC, ENV, COST, REG, SLA)
- Multi-tenant schema with RBAC per workspace
- REST + MCP server (stdio + HTTP) for universal AI assistant integration
- Embeddings layer (centrally hosted by the platform)

## Status

🚧 Building. Not yet usable.
