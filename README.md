# kvendra-platform

**The harness for advanced engineering.**

Multi-tenant KB engine, REST + MCP server, and embeddings layer for the Kvendra platform. The structural core — typed entities, transactional changelog, workspace isolation.

## Status

Pre-alpha. Placeholder repository — implementation begins with the Kvendra Alpha milestone.

## What will live here

- Typed entity engine across `PRJ`, `CMP`, `IF`, `REQ`, `TEST`, `ISSUE`, `REL`, `ROAD`, `ADR`, `PAT`, `GLO`, `STD`, `RUN`, `UX`, `DOC`, `ENV`, `COST`, `REG`, `SLA`.
- Multi-tenant Postgres + pgvector schema with RBAC per workspace.
- REST API + MCP server (stdio + HTTP) for universal AI assistant integration.
- Centrally-hosted embeddings (Bedrock Titan v2 or equivalent) — paid by the platform, not by clients.
